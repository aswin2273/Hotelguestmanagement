A console-based Java application that allows hotel staff to manage guest profiles and stay transactions using JDBC and a relational database.
This system supports complete guest lifecycle management including registration, check-in, check-out with billing, and profile removal with proper transactional handling.

Features

Register Guest
View Guest Details
View All Guests
Check-In Guest (Transactional)
Check-Out Guest (Transactional + Bill Generation)
Remove Guest Record (with Active Stay Validation)

Technologies Used
Java (Core Java)
JDBC
Oracle Database
SQL
Exception Handling
Transaction Management
